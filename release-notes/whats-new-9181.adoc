AFX ストレージ システムで利用できるONTAP 9.18.1 に含まれる新機能について説明します。



== ストレージ リソース管理の機能拡張

[cols="2,4"]
|===
| 更新 | 説明 


| 強化されたボリューム配置  a| 
NetApp AFX は、クラスター内のすべてのノード間でボリュームの配置を自動的に調整します。以前のONTAPリリースでは、配置アルゴリズムはクラスタ内のボリュームの数に基づいていました。アクティビティに関係なく、すべてのノードに同じ数のボリュームが割り当てられます。 ONTAP 9.18.1 以降では、ボリュームを配置または移動するときにノードのパフォーマンスを考慮するようにアルゴリズムが強化されました。これにより、AFX クラスター内のノード間のパフォーマンスバランスが改善され、単一のノードが過負荷になる可能性が大幅に低くなります。
